# 한부모가정 복지 접근성 향상을 위한 입지 분석

## 개요
한부모가구 복지 수요를 행정동 단위로 추정하고,
공간 데이터를 활용해 복지시설 접근성이 낮은 사각지대를 도출한
공간분석 프로젝트를 기록하였다. 

주거지역 기반 dasymetric mapping을 적용하여
행정구역 단위 통계를 실제 거주 가능 지역에 재분배하고,
복지시설 서비스 반경을 기준으로 수요–공급 불균형을 분석하였다. 

## 연구 배경 및 목적
한부모 가정 아동이 겪는 양육 공백과 그로 인한 정서적 어려움 등의 문제를 해결하기 위해 복지시설이 필요한 지역을 선정하고자 한다. 
복지시설 증가의 필요성, 복지 시설의 수요와 공급을 분석하여 서비스 소외지역을 식별한다. 이를 통해 복지시설이 필요한 지역을 제안한다. 
이 repo에서는 주요 코드만 정리하였다. 

## 데이터
- 한국아동청소년 패널조사 2018 (KCYPS 2018)
- 토지이용현황도
- 인구주택총조사 등
※ 원본 데이터는 포함하지 않음

## 분석 방법
1. 동별 인구 비율을 활용해 구 단위 한부모가구 인구를 행정동 단위로 비례 배분
2. 행정동 경계(shp)와 결합해 공간 단위 수요 매핑
3. 주거지역(토지이용 코드 3110, 3120)과 교차하여
   주거 가능 면적 비례로 수요를 재분배(dasymetric mapping)
4. 복지시설 좌표를 기준으로 일정 반경(1,000m)의 서비스 권역 생성
5. 서비스 권역 외 주거지역을 복지 사각지대로 정의하고,
   동별 사각지대 내 한부모가구 수요 밀도 산출

## 주요 결과
- 복지시설 접근성이 낮은 한부모가구 수요 집중 지역 도출
- 행정동 단위 복지 사각지대 비교 및 우선 고려 지역 식별
- 공간 시각화를 통한 수요–공급 불균형 패턴 확인
